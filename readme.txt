Objective of the game is to shoot all the white coins and the red coin into the holes by controlling the motion of the striker.
Game has both mouse controls and keyboard controls. 

Keyboard Controls:

*Use a and c keys to pick the angle at which to hit the striker.
a -> anticlockwise
c -> clockwise

*Use up arrow and down arrow keys to regulate the velocity of the strike

*Use left and right arrow keys to shift the striker to the respective direction

*Hit the spacebar to shoot
